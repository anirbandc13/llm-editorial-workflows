# llm-editorial-workflows

A collection of workflow templates, prompt systems, and research frameworks used for SEO-led, long-form B2B content. These workflows support research, outlining, SERP analysis, and editing, helping maintain clarity, structure, and consistency across technical topics such as AI tools, cybersecurity, cloud hosting, and SaaS.

The prompts and systems in this repository are designed to work with modern LLMs and follow best practices, including explicit instruction hierarchy, context isolation, minimal ambiguity, and controlled output formatting.

## Included
- Outline generation prompts
- Editing and clarity-enhancement workflows
- SERP and keyword analysis frameworks
- Fact-checking structures
- Sample outlines for technical topics
